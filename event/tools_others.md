## VoiceOS — Voice Operating System

- **Website**: https://www.voiceos.com/
- **Background**: YC X25
- **Judge**: Kai Brokering (Founder)

### What It Is

Voice-to-text tool that turns natural speech into polished writing instantly. "Do more, type less."

### Key Features

- **Dictation Mode**: Writes what you meant, not what you said. Auto-removes filler words, fixes grammar.
- **Ask Mode**: Ask VoiceOS and let it write for you (e.g., "Reply I can't but ask to reschedule" → professional email)
- **Privacy**: Data stays on device, audio processed in real time, never stored without permission
- **100+ languages**: Automatic detection, no switching
- Works across all apps: Gmail, Slack, Notion, Google Docs, iMessage, WhatsApp, Cursor, VS Code, Obsidian

### Pricing

| Plan | Price | Features |
|------|-------|----------|
| Free | $0 | 100 uses/week, Dictation + Ask Mode |
| Pro | $12/month | Unlimited, priority support, team features |
| Enterprise | Custom | SOC 2, HIPAA, SSO/SAML |

### Hackathon Relevance

1st place prize に「VoiceOS for 1 Year」含む。90秒ピッチ動画の voice narration に使える（peripheral integration）。Kai Brokering が審査員。
