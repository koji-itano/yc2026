# Real Physical Gigs Tabletop Canister Demo Quick Card v1.0

## Use

This is the shortest live-reference version of the canister demo.

- Use it during the judged demo or while filming
- Pair it with `strategy/tabletop_canister_demo_runbook_v1.0.md`
- Keep it visible on the desk or on the laptop edge

## 20-Second Setup Check

- Canister centered on tape mark
- Cap in `open / fault` state
- `TRACKER A` visible
- Laptop on alert-ready screen
- Phone unlocked with AR or manual fallback ready

## Gold Path

1. Trigger alert on laptop: `Valve #3 pressure anomaly. Physical inspection required.`
2. Worker opens phone task and frames the canister.
3. AR locks to `TRACKER A` and highlights the red cap.
4. Capture `BEFORE`.
5. Turn the red cap clockwise to the closed state.
6. Capture `AFTER`.
7. Phone shows `VERIFIED`.
8. Laptop shows before/after, audit log, and `Payment released`.

## Spoken Cues

| Moment | Line |
|---|---|
| Alert | `The AI detects an anomaly and dispatches a physical inspection.` |
| Target lock | `The phone guides the operator to the exact control point.` |
| Action | `Capturing before, closing the cap, capturing after.` |
| Close | `Now we verify and release payment with an audit trail.` |

## 60-Second Timing

| Time | Action |
|---|---|
| 0-10 sec | Alert on laptop |
| 10-20 sec | Phone acquires target |
| 20-35 sec | Capture `BEFORE` and close the cap |
| 35-45 sec | Capture `AFTER` and show `VERIFIED` |
| 45-60 sec | Show audit log and payout state |

## Fallback Ladder

1. Hold still and reacquire `TRACKER A` for up to 3 seconds.
2. Switch to MindAR if the primary lock fails twice.
3. Switch to manual overlay mode if tracking is still unstable.

Fallback line:
`Tracking is degraded in this lighting, so Real Physical Gigs is switching to guided manual verification without changing the task flow.`

## Hard Rules

- Tabletop only
- Prop only, no real pressure or liquid
- No walking, floor staging, or audience participation
- If the prop shifts, reset and restart from the alert

## Reset

- Reopen cap
- Return canister to tape mark
- Clear prior before/after images
- Return dashboard to alert-ready state
- Wipe glare off canister and phone lens
