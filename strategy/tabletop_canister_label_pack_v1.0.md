# Real Physical Gigs Tabletop Canister Label Pack v1.0

## Use

This file is the printable source for the physical labels used in the tabletop canister demo.

- Print on plain paper or cardstock
- Cut each block separately
- Tape labels firmly so they do not shift during AR tracking or filming
- Keep one spare copy of `TRACKER A`

## Canister Body

```text
VALVE #3
```

## Cap Marker

```text
TURN THIS
```

## Image Target

Print two copies.

```text
TRACKER A
```

Pressure Vessel Demo - Tabletop Only

## Print Notes

- Use black text on white paper for readability from camera distance
- Make `VALVE #3`, `TURN THIS`, and `TRACKER A` the largest labels
- Keep the base card wide enough to read in the opening shot
- If glare is an issue, use matte paper or tape labels onto non-gloss backing
