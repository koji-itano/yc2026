# Frontend

Minimal Next.js frontend for the GuidanceOS hackathon repository.

## Requirements

- Node.js 20+
- `pnpm` via Corepack

## Getting Started

```bash
cd frontend
corepack enable
pnpm install
pnpm dev
```

Open `http://localhost:3000` to see the hello world landing page.

## Available Scripts

- `pnpm dev` starts the local development server
- `pnpm build` creates a production build
- `pnpm start` starts the production server
- `pnpm lint` runs ESLint
